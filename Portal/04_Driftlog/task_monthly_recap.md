# 任务三:写月度回顾报告

## 触发

我让你整理某个月时(如"整理4月"、"做一下4月的Travel Recap")。月份由我指定,可以是上个完整月,也可以是任意历史月份。

## 输出形式

不直接发在对话里,而是在已连接的 Google Drive 中创建 Google Docs 文档。
写完后在对话里只回:文档链接 + 一句简短确认(如"4月的Travel Recap写好了,链接在这里")。
不要把报告内容也复制一份在对话里。

## 文件名(Google Drive 里的文件标题)

格式:
```
[英文月份] Travel Recap
```

例:
- `January Travel Recap`
- `April Travel Recap`

## 内容来源

三个来源拼合使用:
1. project 里的 source 文件(我定期整理的当月流水、长期跟踪的指标数据)
2. 我在对话里当下发给你的补充素材
3. 我已经写过的当月旅行日记(在 Google Drive 里,文件名格式见 task_diary_writing.md)

如果三个来源里关键事件出现冲突,以我当下的话为准,并提醒我是否要更新 source 文件。

## 内容重心

**主体专注外食、外出、旅行**——这是 Travel Recap 的核心。

健身、饮食、工作状态、睡眠等"生活复盘"内容**只在必要时少量带过**,不展开:
- 该月的工作/健身节奏对生活产生明显影响时,可一句带过(如"工作压住整月,周末多被加班吃掉")
- 长期指标(睡眠债务、训练频次等)有显著情况时,可以引用具体数字
- 没有显著情况时,这些内容不写

## 报告结构

固定四块:开头总定性 → 中间分类盘点 → 末尾评估 → 收尾对比句。

### 1. 开头总定性

第一段给整月下判断,落到事实上,不抽象。

句式不固定,跟着该月特性走。简单的一句话标签足够时(如"修复月""硬撑月")就用一句话;复杂的多线状况(如"项目危机+感情内耗+在多线崩溃边缘靠惯性撑过去")就用多个事由+一个底色判断。

**必须包含与上月的对比**——明确说出和上月的关系或递进(如"如果说1月是修复月,2月基本进入高压运转模式")。

### 2. 中间分类盘点

固定三类,每类下用清单形式列具体日期+做了什么+关键细节(吃什么、和谁、去哪)。

**周末/节假日**
列出该月所有周末和节假日,每条注明日期+主要内容。
重点突出外出、旅行、社交、餐厅。
在家加班、纯休息日也要列,但简短。

**平日外食**
列出工作日有外食或外出的日子,每条注明日期+地点+菜系或店名。
保留店名、菜名、和谁吃的这类细节。

**两点一线日子**
"工作日 + 当天除了上班/在家办公和健身房之外没有其他记录性内容"的日子。
健身房算在两点一线内,不破坏定义。
要给出**明确天数和占比百分比**,格式如"X天,占整月Y%(X/月总天数)"。
该月有特殊事件(旅行、生病等)时,可以二次计算"排除X段后的占比"。
列出具体日期(如果不太多)。

### 3. 末尾评估

固定子段:**好的地方 / 差的地方 / 最准确的定性**。

每个子段下用清单列出要点,要点必须落到具体事实,不空喊。

**好的地方**和**差的地方**:从该月的外食/外出/旅行内容里归纳,健身饮食工作只在显著影响生活质量时才提。

**锚点 / 隐患 / 节点**:可选子段,有才写。
- 锚点:这个月真正撑住人的东西
- 隐患:埋下的雷或长期问题
- 节点:有意义的转折或决定性时刻

### 4. 收尾对比句(最准确的定性)

固定结构:三组对比句,"X月不是Y,是Z"的句式。
最后一句可以是判断或观察(不必强行对比),如"后半月明显比前半月更像正常的你"。

例(1月):
- 1月不是成果月,是修复月
- 不是无聊月,是恢复中的生活月
- 后半月明显比前半月更像正常的你

例(3月):
- 3月不是平稳月,是多线同时出问题的压力峰值月
- 不是没在撑,而是撑的方式靠训练和惯性,不是靠状态好
- 月末的断舍离和独饮清酒,是3月真正属于自己的两个节点

## 写作语气

通用写作原则见 principles.md(语气、细节、情绪、不串联、反升华)。

任务三特有的要求:

- 比任务二更直接、更有判断力。这是给我自己看的私人评估,不是给别人看的记录,可以下评价
- 评价要落到事实上(数据、具体事件、具体日期),不空喊
- 不升华、不喊口号、不给鸡汤——核心还是反升华
- 出现具体数字(睡眠分钟数、训练频次、活动人数等)时,**保留原始数字**,不要为了文字流畅而模糊化

## 不要做的事

- 不要把1-3月那三篇里"健身饮食工作睡眠"占比过大的写法当模板。那些是写歪了的版本,任务三要纠正回来——主体是 Travel Recap(外食/外出/旅行),生活复盘内容只在必要时少量带过
- 不要凭空发明素材里没有的事件、数字、感受
- 不要省掉与上月的对比
- 不要省掉两点一线占比的具体百分比
